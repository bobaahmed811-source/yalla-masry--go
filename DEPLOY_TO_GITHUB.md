
# خريطة الطريق الملكية لنقل المملكة إلى GitHub

يا سيدتي، هذه هي بردية الإرشاد التي ستضمن نقل مشروعك (مملكتك) بالكامل وبأمان إلى GitHub. اتبعي هذه الخطوات بدقة.

### المتطلبات الأساسية:
1.  **تثبيت `git`**: تأكدي من أن `git` مثبت على جهاز الكمبيوتر الخاص بك.
2.  **حساب GitHub**: يجب أن يكون لديكِ حساب على GitHub.

---

### الخطوة الأولى: إنشاء مستودع (بيت جديد) على GitHub

1.  اذهبي إلى [github.com/new](https://github.com/new) لإنشاء مستودع جديد.
2.  **اسم المستودع (Repository name)**: اختاري اسمًا للمشروع، على سبيل المثال `yalla-masry-academy`.
3.  **الوصف (Description)**: (اختياري) يمكنكِ إضافة وصف مثل "The Royal Academy for learning Egyptian Arabic".
4.  **خاص أم عام (Private/Public)**: اختاري `Private` إذا كنتِ لا تريدين أن يرى أي شخص الكود.
5.  **الأهم**: **لا تقومي** بتحديد أي من هذه الخيارات:
    -   `Add a README file`
    -   `Add .gitignore`
    -   `Choose a license`
    يجب أن يكون المستودع فارغًا تمامًا.
6.  اضغطي على زر **"Create repository"**.

---

### الخطوة الثانية: ربط مشروعك بالمستودع الجديد ورفع الملفات

بعد إنشاء المستودع، سيُظهر لك GitHub صفحة بها بعض الأوامر. نحن سنستخدم مجموعة الأوامر تحت عنوان `…or push an existing repository from the command line`.

افتحي نافذة الأوامر (Terminal أو Command Prompt) **داخل مجلد مشروعك** `yalla-masry-academy` ونفذي الأوامر التالية بالترتيب، واحدًا تلو الآخر:

1.  **تهيئة المستودع المحلي (إذا لم يكن موجودًا):**
    ```sh
    git init -b main
    ```

2.  **إضافة جميع الملفات لتكون جاهزة للنقل:**
    ```sh
    git add .
    ```

3.  **تسجيل أول "ختم" على هذه النسخة من الكود:**
    ```sh
    git commit -m "First royal decree: The initial foundation of the Kingdom"
    ```

4.  **ربط مشروعك المحلي بالبيت الجديد على GitHub:**
    (قومي بنسخ هذا السطر كما هو من صفحة GitHub التي ظهرت لكِ بعد إنشاء المستودع)
    ```sh
    git remote add origin https://github.com/YourUsername/yalla-masry-academy.git
    ```
    *(استبدلي `YourUsername` باسم المستخدم الخاص بكِ و `yalla-masry-academy.git` باسم المستودع الفعلي)*

5.  **رفع ونقل جميع الملفات إلى GitHub:**
    ```sh
    git push -u origin main
    ```

---

### التأكيد النهائي

بعد تنفيذ الأمر الأخير، قومي بتحديث صفحة المستودع على GitHub. يجب أن تري الآن جميع ملفات ومجلدات المملكة قد ظهرت هناك.

بهذا، تكونين قد نقلتِ المملكة بأكملها بأمان وسلام.
